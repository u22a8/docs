This guide takes you from nothing to a scored piece of text using a model from the public catalog. Authoring your own model is covered in [the REST API reference](/api-reference/introduction).

<Steps>
  <Step title="Get an API key">
    Sign in to the [console](https://u22a8.ai/console) and create a key. Copy the secret — it is shown only once.

    ```bash
    export U22A8_API_KEY="your-key"
    ```
  </Step>
  <Step title="Score a piece of text">
    Score a commit message against the public `u22a8.commit-message` model. The simplest request sends the text as the raw body:

    ```bash
    curl https://u22a8.ai/v1/models/u22a8.commit-message/score \
      -H "Authorization: Bearer $U22A8_API_KEY" \
      -H "Content-Type: text/plain" \
      --data-binary 'Fix race condition in worker dispatch by draining the in-flight queue before shutdown'
    ```
  </Step>
  <Step title="Read the result">
    The default response is JSON. Each trait gets a score from 0–100, alongside a composite, a confidence signal, and headroom to the next tier:

    ```json
    {
      "scores": {
        "intent_clarity": 77,
        "scope_precision": 85,
        "actionable_summary": 84,
        "context_sufficiency": 71,
        "signal_density": 86
      },
      "composite": 80,
      "confidence": "moderate",
      "headroom": 10
    }
    ```

    For a human-readable view, ask for `Accept: text/plain`:

    ```bash
    curl https://u22a8.ai/v1/models/u22a8.commit-message/score \
      -H "Authorization: Bearer $U22A8_API_KEY" \
      -H "Accept: text/plain" \
      -H "Content-Type: text/plain" \
      --data-binary 'Fix race condition in worker dispatch by draining the in-flight queue before shutdown'
    ```

    ```text
    intent_clarity       ███████████████░░░░░   77  ●●○  Developing
    scope_precision      █████████████████░░░   85  ●●●       Solid
    actionable_summary   █████████████████░░░   84  ●●●       Solid
    context_sufficiency  ██████████████░░░░░░   71  ●●○  Developing
    signal_density       █████████████████░░░   86  ●●●       Solid
    ───────────────────  ────────────────────  ───
    Composite            ████████████████░░░░   80  ●●○
    ```
  </Step>
</Steps>

<Prompt description="Score text with an AI assistant">
Using the u22a8.ai REST API, write a small script that scores a piece of text against the `u22a8.commit-message` model and prints each trait score with its tier label. The endpoint is `POST https://u22a8.ai/v1/models/{handle}/score` with an `Authorization: Bearer` header; the body can be the raw text with `Content-Type: text/plain`.
</Prompt>
